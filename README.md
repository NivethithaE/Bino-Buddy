Bino Buddy – Movie Chatbot

Bino Buddy is a WhatsApp-style interactive movie chatbot built with React.js and Tailwind CSS. It helps users discover movies, get cast and crew details, decode storylines, and receive recommendations through a friendly chat interface.

Features

Dynamic chat bubbles for user and bot messages

Real-time responses based on keywords (movie, recommend, help, hi, bye)

Movie recommendations by genre or mood

Storyline decoding and trivia sharing

Smooth animations and interactive UI

Fully responsive design

Tech Stack

Frontend: React.js, Tailwind CSS

State Management: React Hooks (useState, useEffect)

Styling & Animations: Tailwind CSS + CSS animations

Deployment: Vercel (or any React hosting platform)
Usage

Type “hi” or “hello” to get a greeting from Bino Buddy.

Ask about a movie to get cast, crew, or storyline info.

Ask for recommendations by genre or mood.

Type “help” to see a list of things Bino Buddy can do.
